How to Setup the "First!" Game

A step-by-step guide to adding the competitive "First!" widget to your Twitch stream.

Step 1 Configure the Tool

Click the Use Tool button on the main page. You will see a setup screen.

  • Enter your Twitch Channel Name.
  • (Optional) Change the chat command (default is !first).
  • Click "Customize Look" to change the colors or the emoji (ðŸ‘‘).

Step 2 Get the Link

Once you click Start, look at your browser's address bar (or click the "Copy Link for OBS" button inside the tool).

https://.../first.html?channel=yourname&emoji=ðŸ‘‘

Step 3 Add to OBS

  1. Open OBS Studio.
  2. Under Sources, click + and select Browser.
  3. Name it "First Game".
  4. Paste the URL you copied into the URL field.
  5. Set Width: 1920 and Height: 1080.
  6. Check "Refresh browser when scene becomes active".
  7. Click OK.

Admin Commands

If you need to reset the round early (e.g., for testing or if a bot won), type this in your chat:

!resetfirst

*Only Broadcasters and Moderators can use this command.*
