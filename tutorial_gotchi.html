How to Setup "Streamagochi"

A guide to adding your virtual pet overlay, including the special "Peek Mode".

Fixed Mode (Default)

The pet sits at the bottom of the screen with its status bars always visible.

Peek Mode (Recommended)

The pet hides off-screen. It randomly "peeks" in from the sides (every 10-60 mins) or pops up only when it needs something (Hungry/Tired).

Step 1 Basic Setup

  • Click Use Tool on the main page.
  • Enter your Twitch Channel Name.
  • Click Start.
Pro Tip

Step 2 Enabling Peek Mode

By default, the pet is in "Fixed Mode". To enable "Peek Mode", you need to modify the link slightly in OBS.

  1. Copy the URL from the tool after you click Start.
  2. Paste it into a text editor (or directly into OBS).
  3. Add this to the very end of the link: &mode=peek

Standard Link:

.../streamagochi.html?channel=ninja

Peek Mode Link:

.../streamagochi.html?channel=ninja&mode=peek

Step 3 Add to OBS

Add a Browser Source in OBS, paste your link (with or without &mode=peek), and set size to 1920x1080.

Note: If the background isn't transparent, right-click the source in OBS, go to Properties, and click "Refresh cache of current page".

Chat Commands

!feed - Reduces Hunger
!play - Reduces Boredom
!sleep - Restores Energy
!revive - Resets if it dies
